InkClock Web Simulator

预览信息

这是一个基于HTML/JavaScript的InkClock模拟器，可以在浏览器中直接预览固件界面。

功能：

  • 模拟电子墨水屏显示效果
  • 支持不同的页面切换（日历、股票、消息）
  • 支持数字/模拟时钟模式切换
  • 实时显示时间和天气信息
  • 模拟电池电量显示
